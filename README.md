# Clean Pro

Landing page for a professional cleaning service (Hamburg). Built with **React 19**, **Vite 6**, **TypeScript**, and **Tailwind CSS 4**.

## Scripts

- `npm run dev` — development server (default [http://localhost:5173](http://localhost:5173))
- `npm run build` — typecheck and production build to `dist/`
- `npm run preview` — serve the production build locally

## Project layout

- `index.html` — entry HTML and SEO meta tags
- `src/` — React entry (`main.tsx`, `App.tsx`, global styles)
- `components/` — UI sections
- `public/` — static assets (`images/`, `robots.txt`, `sitemap.xml`, manifest)
